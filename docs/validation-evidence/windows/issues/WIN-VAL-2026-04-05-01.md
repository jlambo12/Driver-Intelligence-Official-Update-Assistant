# WIN-VAL-2026-04-05-01 — WMI ACL runbook hardening

- Date opened: 2026-04-05
- Owner: Platform QA
- Severity: Medium
- Source evidence: EV-002 (restricted WMI validation)

## Problem

В restricted WMI окружениях оператору требуется более явный runbook по диагностике namespace ACL, чтобы быстрее отличать policy-driven access denials от реальной поломки runtime.

## Action items

1. Добавить runbook раздел по проверке WMI ACL и expected fallback статусов.
2. Добавить quick-check чеклист для support handoff.
3. Привязать runbook ссылку в release checklist.

## Exit criteria

- Runbook добавлен и утверждён QA + Runtime owner.
- Повторный restricted-WMI прогон выполняется без эскалации как инцидент.
